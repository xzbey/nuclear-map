# Карта ядерных ударов

## Возможности
* просмотр испытательных ядерных ударов
* просмотр реально нанесенных ядерных ударов
* фильтр типа удара
* фильтр по году
* фильтр по стране, осуществившей запуск
* масштабирование карты
* выбор карты

### Просмотр испытательных и реально нанесенных ударов
При запуске веб-версии карты подгружаются точки ударов всех стран до 2009 года. Удары объединены по стране и показываются изнчаально как точка с количеством. 
При наведении на маркер показывается страна. При нажатии на маркер с количеством масштаб увеличивается, показывая больше данных.

Иконки у ударов разные - по стране. При нажатии на иконку удара показывается дополнительная информация:
* серия
* название удара
* дата
* время
* место падения
* тип удара
* цель
* мощность: минимальная и максимальная 
* размер кратера
* название боеголовки
* спонсор/цель
* cтрана, осуществившая запуск
* тип удара: испытательный или реальный

_Если какой-либо доп. информации нет - то она неизвестна._

### Фильтр по типу удара
В левой верхней части страницы находится выпадающий список с типом удара, фильтрующий точки. Показывает либо все удары, либо только испытательные (test), либо только реально нанесенные (strike).

### Фильтр по году
В левой верхней части страницы находится ползунок выбора года, который фильтрует точки и показывает все удары до выбранного года (включая). Минимальный год - 1944, максимальный - 2009.

### Фильтр по стране
В левой верхней части страницы находится выпадающий список с странами, фильтрующий точки. Он показывает либо все удары, либо какой-то конкретной выбранной страны.

### Масштабирование карты
В левой верхней части страницы находятся 2 кнопки увеличения и уменьшения масштаба карты. Для некоторых карт, максимальный зум меньше, чем для других. Например, для `Satellite` максимальный зум - 19, а для `Topo` - 17.

### Выбор карты
Веб версия карты предусматривает изменение карты на другие, а именно:
* OpenStreetMap
* CARTO Dark
* Esri Satellite
* OpenTopoMap
* Google **[Отключено - Лицензирована]**
* Google Satellite **[Отключено - Лицензирована]**
* Yandex **[Отключено - Лицензирована]**

**Карту можно изменить нажав на кнопку соответствующей карты в левом нижнем углу.**

Также карту можно изменить, открыв консоль и написав для:

* OpenTopoMap : `console.log('topo')` **[Часто работает только с VPN]**
* Google : `console.log('google')` **[Отключено - Лицензирована]**
* Google Satellite : `console.log('google satellite')` **[Отключено - Лицензирована]**
* Yandex : `console.log('yandex')` **[Отключено - Лицензирована]**


## Подробности
Точки были получены в результате парса с сайта `johnstonsarchive.net`, конкретно https://johnstonsarchive.net/nuclear/tests/index.html в разделе **"Database of nuclear tests, full version"**. 

Сначала были извлечены конкретные столбцы информации в таблицы формата `csv` в `src/db/csv` с помощью написанной программы `src/py/parser.py`. После этого данные из таблицы были извлечены, обновлены сокращения и конвертированы в файлы формата `geojson` в `src/db/geojson` с помощью написанной программы `src/py/converter.py`.

Была создана карта веб формата с помощью библиотеки `Leaflet.js`. Карта имеет дефолтную систему координат Leaflet - Web Mercator (EPSG:3857). Далее были загружены точки с данными из `src/db/geojson`. Реализованы функции показа и фильтрации точек, а также смены карты.
